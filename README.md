# Museu Musical
 É um projeto de um museu contendo a história da música, com possibilidade de viajar entre diferentes épocas, interação com albúns e também a possibilidade de escutar as músicas enquanto navega entre uma sala e outra, diferentes estilos, artistas com o objetivo de ensinar e mostrar a origem musical no mundo.O sistema permite que o usuário navegue por "Salas Virtuais" que representam diferentes épocas da humanidade, permitindo a interação com álbuns icônicos e o aprendizado sobre a origem de estilos musicais, desde a música clássica até as trilhas sonoras (soundtracks) contemporâneas de Games, Animes e Cinema.
